# Чек-лист API тестирования OWASP Juice Shop (Postman)

## Регистрация (POST /api/Users)
- [x] POST /api/Users возвращает 201
- [x] Email сохраняется корректно
- [ ] Регистрация с существующим email → 409
- [ ] Пустой email → 400
- [ ] Пароль < 5 символов → 400

## Авторизация (POST /rest/user/login)
- [x] POST /rest/user/login возвращает 200 с валидными данными
- [x] В ответе присутствует authentication.token
- [ ] Невалидный пароль → 401
- [ ] Несуществующий email → 401

## Корзина (Basket)
- [x] POST /api/BasketItems добавляет товар (201)
- [x] GET /rest/basket/{id} показывает корзину (200)
- [ ] Без токена авторизации → 401
- [ ] Несуществующий ProductId → 400

## Заказы (Orders)
- [x] POST /rest/basket/{id}/checkout оформляет заказ (200)
- [x] GET /rest/order-history показывает историю (200)
- [ ] Оформление пустой корзины → 400
- [ ] Повторный заказ той же корзины → 409

## Переменные окружения (Postman)
- [x] {{host}} работает
- [x] {{email}} передается между запросами
- [x] {{token}} автоматически сохраняется
- [x] {{basketId}} сохраняется и используется

## Collection Runner
- [x] Все запросы запускаются подряд
- [x] Нет ошибок при последовательном выполнении
- [x] Результаты экспортируются (JSON/HTML)
