TC-API-04: Просмотр содержимого корзины

Предусловия:
1. В корзину добавлен товар (TC-API-03)
2. Переменная {{basketId}} содержит ID корзины

Шаги:
1. Отправить GET запрос на {{host}}/rest/basket/{{basketId}}
2. Headers: Authorization: Bearer {{token}}

Ожидаемый результат:
- Статус код: 200 OK
- В ответе массив Products
- Количество товаров > 0
