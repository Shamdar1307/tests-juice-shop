TC-API-06: Просмотр истории заказов

Предусловия:
1. Оформлен хотя бы один заказ (TC-API-05)
2. Пользователь авторизован

Шаги:
1. Отправить GET запрос на {{host}}/rest/order-history
2. Headers: Authorization: Bearer {{token}}

Ожидаемый результат:
- Статус код: 200 OK
- В ответе массив заказов
- Массив не пустой
