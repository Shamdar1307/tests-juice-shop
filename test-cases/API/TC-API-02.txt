TC-API-02: Авторизация и получение JWT токена

Предусловия:
1. Пользователь зарегистрирован (TC-API-01)
2. Переменная {{email}} содержит email пользователя
3. Переменная {{password}} = "Test123!"

Шаги:
1. Отправить POST запрос на {{host}}/rest/user/login
2. Body (raw JSON):
   {
     "email": "{{email}}",
     "password": "{{password}}"
   }

Ожидаемый результат:
- Статус код: 200 OK
- В ответе присутствует поле authentication.token
- Токен не пустой
- ID корзины (bid) сохраняется в переменную {{basketId}}
