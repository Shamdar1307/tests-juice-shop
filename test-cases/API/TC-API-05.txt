TC-API-05: Оформление заказа из корзины

Предусловия:
1. В корзине есть товары (TC-API-04)
2. Пользователь авторизован

Шаги:
1. Отправить POST запрос на {{host}}/rest/basket/{{basketId}}/checkout
2. Headers: Authorization: Bearer {{token}}
3. Body: пустой

Ожидаемый результат:
- Статус код: 200 OK
- Заказ создан
