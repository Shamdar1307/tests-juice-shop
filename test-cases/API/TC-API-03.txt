TC-API-03: Добавление товара в корзину

Предусловия:
1. Пользователь авторизован (есть {{token}})
2. Товар с ID=1 существует

Шаги:
1. Отправить POST запрос на {{host}}/api/BasketItems
2. Headers: Authorization: Bearer {{token}}
3. Body (raw JSON):
   {
     "ProductId": 1,
     "quantity": 2
   }

Ожидаемый результат:
- Статус код: 200 OK (или 201 Created)
- В ответе данные добавленного товара
- Товар добавлен с указанным quantity
